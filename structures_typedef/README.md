# 0x0D. C - Structures, typedef

## Description
This project covers the basics of structures (`struct`) and type definitions (`typedef`) in C programming. It includes defining custom data types, initializing them, printing their content, and managing dynamic memory allocation for structures.

## Files
| File | Description |
| --- | --- |
| `dog.h` | Header file containing the definition of `struct dog`, `dog_t`, and function prototypes. |
| `1-init_dog.c` | Function that initializes a variable of type `struct dog`. |
| `2-print_dog.c` | Function that prints a `struct dog`. |
| `4-new_dog.c` | Function that creates a new dog (dynamic memory allocation). |
| `5-free_dog.c` | Function that frees the memory allocated for a `dog_t`. |

## Requirements
* Compiled on Ubuntu 20.04 LTS using `gcc`
* Compiler flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
* No memory leaks (verified using `valgrind`)
